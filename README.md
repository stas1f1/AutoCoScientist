# AutoCoScientist

![Python 3.12+](https://img.shields.io/badge/python-3.12+-blue.svg)
![License](https://img.shields.io/badge/license-BSD%203--Clause-green.svg)
![LangGraph](https://img.shields.io/badge/built%20with-LangGraph-orange.svg)

**AutoCoScientist** is an agentic framework designed for automatic conversion of data-driven experiments into workflows and tools usable by LLMs. It incorporates pygrad library to understand and utilizize available repositories through a graph-based documentation system.
